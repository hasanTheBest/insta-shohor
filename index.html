Javascript Conceptual Topics

1. How does JavaScript Work?

JavaScript runs in browser environment powered by an engine. The V8 is the most popular JS engine which is developed by Google and used by Chrome and NodeJs.

Browser engine’s take the raw JS code and parse line by line in the direction top to bottom and produce an Abstract Syntax Tree (ABS). If the given code does not follow proper syntax it throws the syntax error at this stage.

The engine ignites interpreter to interpret Syntax Tree and produce ByteCode (machine code) which is appropriate to understand by the machine. Machine runs the piece of code and JavaScript performs it task.

In modern engine like V8 performs Just In Time (JIT) and optimize ByteCode to produce faster to run and efficient machine code.
